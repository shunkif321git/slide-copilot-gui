# slide-copilot-gui

講義スライドの作成を支援するGUIアプリです。
PythonとTkinterで構築され、ユーザーが講座情報・章構成を入力すると、自動でスライドのテンプレートを作成します。

---

## 必要環境

- Pyton 3.8以上
- pip

---

## 実行方法

bash
以下のコードを実行してください
git clone https://github.com/shunkif321git/slide-copilot-gui.git
cd slide-copilot-gui
pip install -r requirements.txt
python slide_gui.py
